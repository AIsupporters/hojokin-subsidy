# 補助金調べるくん（公開静的サイト）

国・大阪府の補助金/助成金の新着をまとめて表示する無料の静的サイト。運営：AIサポーターズ株式会社。

このリポジトリは**公開する静的ファイルのみ**を管理します（ビルド不要・サーバー不要）。

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` | ページ本体 |
| `styles.css` | スタイル |
| `app.js` | 一覧・絞り込み・登録フォーム・出し分けの処理 |
| `eligibility.js` | 業種×従業員規模 → 小規模/中小の目安・補助上限帯（確定データのみ） |
| `data/subsidies.json` | 公開用の補助金データ（事前生成） |
| `data/overrides.json` | 締切/補助額の一次情報上書き（ビルド入力） |

## デプロイ

静的サイトとしてそのまま配信（Render の Static Site / Cloudflare Pages 等）。ビルドコマンド不要、公開ディレクトリ＝このフォルダ。

## データ更新

`data/subsidies.json` は親プロジェクトの `scripts/build-site-data.mjs`（Google Sheets 台帳を読み取り→整形）で再生成し、本リポジトリにコミットして反映する。サイトはこの静的JSONを読むだけ（実行時に台帳へはアクセスしない）。

## 注意

- 表示する重要度・該当度・規模判定は機械的な目安であり、申請の可否・採択の助言ではない。補助額・補助率・締切は各制度の公式公募要領で要確認。
- 認証情報・台帳ID等の内部情報は本リポジトリに含めない。
